Build tools:
------------
Roles of Build tools:

									1.Creating a project structure
									2.Adding required depedencies in build path
									3.Creating packages for 
											1.Source code
											2.Test code
									4.Compile all source code + test code 
									5.Run all test cases
									6.Creating package(jar/war) for current project 

Maven:
=====
Maven is a build automation tool used primarily for Java projects. 

Maven can also be used to build and manage projects written in C#, Ruby, Scala, and other languages. 

The Maven project is hosted by the Apache Software Foundation, where it was formerly part of the Jakarta Project.
